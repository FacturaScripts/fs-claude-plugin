El widget Subcuenta permite seleccionar una subcuenta desde el listado de cuentas, donde podemos buscar y filtrar por ejercicio.

```
&lt;column name=&quot;codsubcuenta&quot; numcolumns=&quot;3&quot; order=&quot;110&quot;&gt;
	&lt;widget type=&quot;subcuenta&quot; fieldname=&quot;codsubcuenta&quot;/&gt;
&lt;/column&gt;
```

Al hacer click en el widget, se abrirá una nueva interfaz la cual servirá para para encontrar la subcuenta que queramos asignar.

![Imagen del widget subcuenta](https://i.imgur.com/dzMkWNT.png)

Dentro de esa misma interfaz se podrán filtrar las subcuentas por codsubcuenta o por descripción desde el buscador, tambien hay un desplegable para elejir el año y otro para cambiar el orden del listado.

![selector subcuenta](https://facturascripts.com/MyFiles/2025/11/3430.png?myft=4a9a654b9523cd4907b8b4a476cafa9ce8fed8a7)
