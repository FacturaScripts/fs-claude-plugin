Hoy hemos comentado la lista de mejoras ya completada para FacturaScripts 2018.11, siendo el soporte de extensiones el más destacado.

## Lista de mejoras ya completadas
- Soporte de extensiones para controladores y modelos.
- Soporte de extensiones para los XML de las tablas y los XMLView.
- Mejoras en el asistente de configuración inicial para poder indicar el nombre corto y el CIF/NIF de la empresa.
- Mejorada las interfaces de listados al visualizarse desde smartphone.
- Los grupos de los XMLView ahora se pueden alinear verticalmente.
- Nueva barra de debug para sustituir a phpdebugbar, que tenía bastantes problemas de rendimiento.
- Nuevo manejador de errores personalizado para producción.
- Nuevo manejador de errores para desarrollo: whoops.
