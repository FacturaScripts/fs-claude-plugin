Ya se pueden **recomprar** [cursos](https://facturascripts.com/cursos) y **transferirlos** a otros usuarios. Ahora puedes **comprar varias veces** un curso y **transferirlo** a otro empleado o cliente. De este modo puedes centralizar las compras y se evita el engorro de que cada empleado o cliente deba comprar los cursos que necesita.

## ¿Cómo funciona?
- Si ya has aprobado un curso, ahora aparece un **botón** para recomprarlo.
- Si todavía no te has examinado, aparece un **botón transferir**. Al pulsarlo te solicita el email del usuario al que quieres transferirlo.
