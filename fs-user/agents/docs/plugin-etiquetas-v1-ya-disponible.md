Este plugin, ya disponible para FacturaScripts 2020, nos permite [imprimir etiquetas](https://facturascripts.com/plugins/etiquetas) para los productos **desde los albaranes o facturas de compra**. Soporta el formato **[APLI 10490](https://amzn.to/2VmuMHQ)** (A4, 68 etiquetas) y tienes las siguientes opciones:
- **Fila y columna inicial**. Si ya hemos impreso 20 etiquetas de otro albarán, podemos imprimir en la misma hoja indicando que empiece más abajo. Así aprovechamos al máximo las hojas.
- Imprimir **códigos de barra**.
- **Número de etiquetas** por producto.

## Funcionamiento
El funcionamiento es el siguiente, desde un albarán o factura de compra hacemos clic en la pestaña etiquetas, seleccionamos las opciones deseadas y pulsamos el botón imprimir. A continuación introducimos el [papel APLI 10490](https://amzn.to/2VmuMHQ) en la impresora y pulsamos imprimir.
