Ya está disponible la **versión 0.4** del plugin de migración para FacturaScripts 2018. Este plugin se instala en FacturaScripts 2018 y permite migrar los datos de las antiguas tablas de FacturaScripts 2017, como son: artículos, combinaciones, epigrafes, etc...

## Cómo realizar la migración
- Haga una **copia de seguridad** de su base de datos e instalación de FacturaScripts 2017.
- Instale **FacturaScripts 2018** conectando a la misma base de datos de 2017.
- Instale el plugin [FS2017Migrator](https://facturascripts.com/plugins/FS2017Migrator).
- Haga clic en el menú Administrador &gt; Panel de control &gt; Migrador 2017.
- Pulse el botón ejecutar.
