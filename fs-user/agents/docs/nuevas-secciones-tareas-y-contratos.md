Esta semana hemos añadid dos nuevas secciones a la web: **tareas** y **contratos**. Podéis encontrarlas en la página de inicio (siempre que hayáis iniciado sesión).

## Tareas
En la **sección tareas** podréis crear tareas para cada uno de vuestros plugins, para poder organizar vuestro tiempo o coordinaros con otros. Las tareas tienen una descripción en formato markdown, por lo que podéis usar negritas, títulos, enlaces, listas e insertar código de forma muy visual. Además de la descripción tienen estado, asignado, fecha de inicio, duración y demora, para saber cuanto os desviáis.

Es una primera versión y ya estamos organizando las tareas de desarrollo de FacturaScripts desde aquí, en lugar de trello. Pronto añadiremos una vista global para poder ver de forma más agrupada las tareas pendientes de todos los plugins (y el core). Esperamos que nos ayude a todos a tener claro a dónde se dirige el core y cada plugin, y los plazos.

## Contratos
¿Necesitas un servicio o desarrollo a medida sobre FacturaScripts y no tienes el tiempo o los conocimientos necesarios? Ahora puedes usar la **sección contratos** para solicitar estos servicios o desarrollos a **otros partners** y **programadores**. Simplemente pulsa el botón nuevo y dinos qué necesitas.

### ¿Quién puede solicitar estos servicios o desarrollos?
Cualquier cliente, partner o programador. En definitiva, cualquiera que inicie sesión en la web.

#### ¿Hay que pagar?
Por supuesto que tendrás que pagar por esos servicios o desarrollos **que solicitas**. El software es libre, el tiempo de los partners o programadores no.

### ¿Quién puede responder a estas solicitudes?
Todo el que **cumpla los requisitos** necesarios. Si por ejemplo se solicita un desarrollo a media sobre FacturaScripts 2017, el requisito para poder contestar a la oferta será haber superado el [curso básico de programación (versión 2017)](https://facturascripts.com/cursos/curso-basico-de-programacion-version-2017).

**Además tendrás que unirte** al [equipo de contratos](https://facturascripts.com/colabora/CONTRACTS) para que te lleguen las ofertas.

#### ¿Qué contesto?
Como partner o programador contestarás si puedes o no puedes realizar el trabajo, cuánto tiempo te llevaría y cuál es el precio.

#### ¿Esto es público?
**No**, las ofertas solamente son visibles por el [equipo de contratos](https://facturascripts.com/colabora/CONTRACTS). Y **las respuestas son privadas** entre el autor y el partner o programador que contesta.

#### ¿Hay que pagar comisión?
Si, habrá una pequeña comisión a pagar **si finalmente realizas el trabajo**. La comisión será visible bajo la descripción del contrato, por lo que podrás verla antes de contestar.
