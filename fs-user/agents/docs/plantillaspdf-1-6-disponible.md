La versión 1.6 del plugin incluye una **nueva plantilla**, la plantilla 4, y nuevas opciones de personalización:
- **Nueva posición para el logotipo**: tamaño completo. El logotipo se coloca centrado en la parte superior sin ningún elemento más (ni empresa, dirección, etc).
- **Altura reservada para las líneas**: ahora podemos definir una altura reservada para las líneas de facturas, albaranes, etc. De esta forma los totales quedarán a la altura que deseemos.

## Estas opciones se unen a las ya incluidas anteriormente
- Tamaño: A3, A4, A5.
- Orientación: vertical u horizontal.
- Logotipo, tamaño y posición.
- Colores.
- Fuente y tamaño.
- Ocultar dirección de envío, observaciones o totales.
- Mostrar numero2.
- Columnas de las líneas, alineación y tipos.
- Texto final (LOPD), texto de agradecimiento y texto de pié de página.
