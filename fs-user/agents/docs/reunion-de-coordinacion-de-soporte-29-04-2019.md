Como ya se comentó, hoy lanzamos actualización de FacturaScripts 2018. Actualización que incluye las nuevas secuencias de documentos (ver [lista de cambios de la actualización](https://facturascripts.com/publicaciones/facturascripts-2018-019-ya-disponible) para más detalles).

## Migración de datos de 2017
Con esta actualización ya se puede utilizar el plugin [FS2017Migrator](https://facturascripts.com/plugins/FS2017Migrator) para realizar la migración de datos de FacturaScripts 2017 a la nueva estructura. Obviamente, al ser una primera versión, se esperan bugs. Es recomendable siempre hacer un backup de la base de datos antes de nada.

## Problemas con las formas de pago
En las primeras versiones de FacturaScripts 2018 no se estaba guardando el vínculo de las formas de pago con la empresa, por eso es posible que los que actualicen se encuentren con mensajes de error al visualizar la formas de pago.

### Solución
La solución es ir a la ficha de la forma de pago y pulsar guardar.

## Plugins de pago
Los primeros plugins de pago se lanzarán en Mayo. No es necesario comprar las versiones de 2018 si ya se tienen las de 2017. También os recordamos que plugins como editar_facturas o CRM ahora están integrados en FacturaScripts 2018. Y que otros como factura_detallada se abandonan.
