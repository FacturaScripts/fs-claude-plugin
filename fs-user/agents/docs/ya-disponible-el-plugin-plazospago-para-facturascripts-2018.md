El [plugin PlazosPago](https://facturascripts.com/plugins/plazospago) permite crear **formas de pago fraccionadas**, como la típica de 50% al inicio y 50% al final, o la de **30-60-90**: un primer recibo a los 30 días, el segundo a los 60 y el tercero a los 90.

Aquellos que hayan comprado el plugin tesorería, y no tengan la clave de actualización caducada, podrán obtener este plugin totalmente gratis en unos días. Os mantendremos informados.
