Ya se puede descargar, desde la [sección descargar](https://facturascripts.com/descargar), la versión **2018.020** de FacturaScripts. O bien utilizar el actualizador (menú Administrador &gt; Panel de control &gt; Actualizador).

## Mejoras en presupuestos, pedidos, albaranes y facturas
- **Mejorado el diseño** y disposición de los elementos de la pestaña principal de presupuesto, pedido, etc...
- Ya se puede modificar el **número 2** o **número de proveedor** desde la pestaña principal al crear o editar.
- Las **facturas rectificativas** ahora muestran enlaces a la factura original y viceversa.
- Al crear un nuevo presupuesto, pedido, albarán o factura, y seleccionar el cliente o proveedor, se selecciona automáticamente la **serie predeterminada** para ese cliente o proveedor.
- Ya se pueden **agrupar presupuestos**, pedidos, etc... desde los listados.
- Los listados ahora muestran el **estado del presupuesto**, pedido, etc... en texto: nuevo, aprobado, cancelado.
- Se han añadido **más colores** para cada tipo de estado.
- Desde la ficha de una serie podemos ver todas las **secuencias asociadas**.

![editar pedido FacturaScripts 2018](https://i.imgur.com/bH40bSL.png)

## Correcciones
- Ahora todos los formularios previenen el envío múltiple, por ejemplo, cuando el ratón del usuario está roto y hace doble o triple clic.
- Al **agrupar presupuestos**, pedidos, etc... se ha corregido el bug que dejaba abiertos los documentos originales. Es decir, ahora si agrupamos dos albaranes en una factura, los albaranes pasan a estado aprobado y no editables (antes se quedaban abiertos).
- Al crear un **nuevo usuario** el selector de página de inicio permanece oculto, solamente se muestra cuando el usuario ha sido creado y ya hay páginas entre las que elegir.

## Actualizar desde la versión 2017
FacturaScripts 2017 todavía tiene más plugins que la versión 2018. Por ese y otros motivos no hemos modificado el actualizador de FacturaScripts 2017 para poder pasar a la nueva versión.

Si quieres probar, te recomendamos los siguientes pasos:
- [Haz una copia de seguridad](https://facturascripts.com/doc/2/copias-de-seguridad/como-hacer-una-copia-de-seguridad-de-facturascripts).
- **Instala FacturaScripts 2018** usando la misma base de datos.
- Instala el [plugin FS2017Migrator](https://facturascripts.com/plugins/FS2017Migrator).

También puedes probar con una [instalación de pruebas de FacturaScripts](https://facturascripts.com/CloudManager) en nuestra nube privada.
