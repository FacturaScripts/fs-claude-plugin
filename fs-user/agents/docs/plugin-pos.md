Saludos, propuesta de plugin POS genérico, se aceptan comentarios, sugerencias, dudas, errores, etc.

Agrega el menu:
	
**point-of-sale**
* cash-registers
* cashups
* opciones
* point-of-sale.

**Primero pasos.**

1. Configurar terminales en el menu **cash-registers**.
2. Confgurar opciones generales en el **menu opciones**. 
3. Acceder a la  opcion **point-of-sale** e iniciar un arqueo.
