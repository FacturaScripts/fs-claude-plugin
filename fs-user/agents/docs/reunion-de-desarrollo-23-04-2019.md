Hoy hemos comentado los desarrollos ya completados para la actualización de esta semana:
- Nuevas **secuencias de documentos**.
- Poder especificar una **subcuenta** distinta para cada **impuesto**.
- Función **redirect()** disponible en la clase Controller.
- Configuración por defecto para **México**.
- Añadido el **IGIC** a la lista de impuestos por defecto para **España**.

## Nuevas secuencias de documentos
La nueva funcionalidad ya implementada permite especificar un **código de numeración** para cada **tipo de documento**, ejercicio o **serie**. A la vez que permite especificar en qué número comenzar. Desde el **menú Administrador** &gt; **Secuencias de documentos** ahora podemos ver y modificar estas secuencias.

## Subcuentas distintas por impuesto
Al igual que en FacturaScripts 2017, ahora es posible especificar una subcuenta distinta para cada impuesto. Simplemente vamos a la ficha del impuesto (en contabilidad &gt; impuestos &gt; impuestos) y rellenamos seleccionamos una subcuenta. Si no es especifica una subcuente se utiliza la subcuenta predefinida en el plan contable.
