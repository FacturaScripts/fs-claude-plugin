Para cancelar una factura o realizar una rectificación o devolución, sigue estos pasos:

1. **Abre la factura** que deseas modificar.
2. Dirígete a la pestaña **Devoluciones**. Allí podrás indicar las cantidades a devolver de cada línea.
3. Una vez que hayas introducido las cantidades, podrás **generar la factura rectificativa**.
4. Tambien se puede eligir el estado en el cual se creara la factura Rectificativa
