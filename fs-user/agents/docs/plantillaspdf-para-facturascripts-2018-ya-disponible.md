[El plugin PlantillasPDF](https://facturascripts.com/plugins/plantillaspdf) permite configurar o personalizar los diseños PDF de facturas, albaranes, pedidos y presupuestos. En esta primera versión podemos elegir entre tres plantillas prediseñadas. Y para cada una de estas plantillas podemos configurar: **tamaño del papel**, orientación, **logotipo**, su tamaño y posición, **colores**, la fuente, su tamaño y color, **las columnas a mostrar** en las líneas de facturas, albaranes, etc. Su alineación y tipo: texto, número, moneda o porcentaje...

Además estas opciones se pueden personalizar para cada tipo de documento. Por ejemplo, puedes hacer que en los albaranes solamente se muestren las cantidades, referencia y descripción, pero no los precios.
