Los transportistas nos permiten controlar con qué agencia enviamos nuestros pedidos, lo cual es muy útil para realizar seguimientos. Para gestionar nuestros transportistas, debemos ir a **Almacén** &gt; **Transportista**. Allí aparecerán las agencias que tengamos registradas.

![Nuevo transportista](https://i.imgur.com/2z35z3h.gif)

### Creación de un nuevo transportista

Para crear un transportista, debes hacer clic en el **botón nuevo** y completar los campos requeridos. Los campos son los siguientes:

* **Código:** Introduce un código para identificar al transportista. *(Opcional, el sistema generará uno en caso de no especificarlo)*
* **Nombre:** Proporciona un nombre identificativo del transportista.
* **Teléfono:** Indica el número de contacto del transportista. *(Este campo es opcional)*
* **Dirección web:** Especifica la dirección web del transportista. *(Este campo es opcional)*
* **Activo:** Indica si el transportista está disponible para su uso.

Una vez que hayas completado todos los campos necesarios, haz clic en el botón **Guardar**. Tu nuevo transportista estará creado y listo para ser utilizado en todos tus documentos de venta.
