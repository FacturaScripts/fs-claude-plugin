Esta actualización, además de ser compatible con el nuevo **plugin de migración**, cuenta con las nuevas **secuencias de documentos**, que permiten tanto personalizar en qué número comienzan las facturas, albaranes, etc... como la codificación de las mismas.

## Migración de datos de FacturaScripts 2017
Ya está disponible la primera versión del plugin [FS2017Migrator](https://facturascripts.com/plugins/FS2017Migrator). Plugin para FacturaScripts 2018 que permite migrar los datos de 2017 a la nueva estructura.

## Secuencias de documentos
Con esta nueva función ya puede:
- Personalizar el número de la primera factura, albarán, pedido o presupuesto.
- Personalizar la codificación, por defecto 2019A000001, pero puede cambiarlo por lo que usted desee.

Puede encontrar más detalles en la [documentación de secuencias](https://facturascripts.com/doc/1/configuracion/secuencias-de-documentos).

![secuencias de documentos](https://i.imgur.com/QaOKt8F.png)

## Mejoras
- Se han mejorado los listados de las **fichas de cliente** y **proveedor** para incluir buscador y más opciones de ordenación. Es decir, ahora podemos ver el listado de facturas de un cliente, desde la ficha del cliente, y hacer búsquedas sobre ese listado.
- Añadida configuración estándar para **Uruguay**.
- Añadida configuración estándar para **México**.
- Añadidas traducciones para Español de **Perú**.
- Añadidos los impuestos **IGIC** a la lista de impuestos por defecto para España.

## Dudas frecuentes
- ¿Tengo que comprar de nuevo los plugins para esta versión?
	- No, a menos que le haya caducado la clave de actualización o haya infringido la licencia.
- ¿Cuando estarán disponibles los plugins para la nueva versión?
	- Siguen en desarrollo. No hay fechas. Los primeros en llegar serán tesorería y plantillas PDF, pero no hay una fecha concreta.
- ¿FacturaScripts 2017 se actualiza automáticamente a la nueva versión?
	- No. No están terminados los plugins, no tendría sentido romper miles de instalaciones.
