En esta actualización se ha añadido la opción &quot;Ocultar direcciones de envío&quot;, para ocultar las direcciones de envío en facturas, albaranes, etc.

![ocultar direcciones de envío](https://i.imgur.com/hbLeS9a.png)

## Ocultar totales
También se ha añadido la opción &quot;Ocultar totales&quot; en los formatos de impresión, para cuando queremos imprimir pedidos sin valorar o similar.

### Ejemplo
Para crear un formato de impresión &quot;Pedido sin valorar&quot;, nos vamos al menú Administrador &gt; Plantillas PDF, hacemos clic en la sección &quot;Formatos de impresión&quot; y pulsamos el botón nuevo:

![formatos de impresión](https://i.imgur.com/ga6sXS1.png)

A continuación creamos el formato &quot;sin valorar&quot; para &quot;pedidos de cliente&quot;, y marcamos la opción de ocultar totales:

![ocultar totales](https://i.imgur.com/stL5unb.png)

## Actualizar el plugin
Para actualizar este o cualquier plugin, solamente debes ir al menú Administrador &gt; Panel de control &gt; Actualizador.
