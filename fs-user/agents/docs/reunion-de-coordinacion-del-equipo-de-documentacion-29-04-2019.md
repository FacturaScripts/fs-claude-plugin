En la reunión de hoy hemos revisado la documentación para ver las carencias. Estas son las tareas a realizar:
- Incluir una imagen del instalador en el apartado de [instalación desde el ZIP](https://facturascripts.com/doc/1/instalacion-de-facturascripts-2018/instalacion-desde-el-zip).
- Añadir a la [sección instalación](https://facturascripts.com/doc/1/instalacion-de-facturascripts-2018) una página titulada **El config.php** donde se detalle para qué sirve este archivo, qué constantes tiene y para qué sirven.
- Vincular la referencia al config.php de la página [instalación desatendida](https://facturascripts.com/doc/1/instalacion-de-facturascripts-2018/instalacion-desatendida) con la nueva página **el config.php**.
- Añadir imagen del login de FacturaScripts sin el CSS para ilustrar la sección **página en blanco** de [problemas frecuentes de la instalación](https://facturascripts.com/doc/1/instalacion-de-facturascripts-2018/problemas-frecuentes). Esto se puede simular eliminando el archivo .htaccess
- Corregir la página de [régimen IVA](https://facturascripts.com/doc/1/configuracion/regimen-iva) para especificar que cada empresa, cliente o proveedor puede tener un régimen de IVA. Como está redactada parece que es algo exclusivo de la empresa.
- Modificar la [página de divisas](https://facturascripts.com/doc/1/configuracion/divisas) para indicar correctamente lo que son las tasas de conversión. Las tasas de conversión se refiere a la conversión a euros, exclusivamente.

## Cambios generales
Cambiar cualquier mención a &quot;botón verde con el simbolo +&quot; por &quot;botón nuevo&quot;.
