[El plugin RemesasSEPA](https://facturascripts.com/plugins/remesassepa) añade gestión de **Remesas bancarias** en formatos **SEPA** CORE, COR1 y B2B. Con él podemos agrupar fácilmente los recibos pendientes de facturas de cliente en remesas, generar el fichero xml y subirlo al banco.

## Características
- Podremos crear remesas desde el **menú Contabilidad** &gt; **Remesas**.
- En cada remesa podremos añadir los recibos que queramos, de entre los recibos pendientes de pago, independientemente de la forma de pago. Si el recibo todavía no está pagado, podemos añadirlo a la remesa.
- Podemos **agrupar los recibos por cliente**, generando una única entrada por cliente con el importe total de los recibos de ese cliente.
- Podemos generar tanto remesas SEPA CORE, como COPR1 y B2B.

## ¿Ya tienes el plugin Tesorería?
Si ya tienes el plugin tesorería para FacturaScripts 2017 puedes obtener el plugin RemesasSEPA sin coste adicional. Ve a tu lista de claves, haz clic en la de tesorería y pulsa el botón 2018.
