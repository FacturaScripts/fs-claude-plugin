Hoy hemos repasado la lista de cambios para la **versión 2018.06**, que esperamos lanzar a final de mes.

## Tareas en desarrollo
Estas tareas ya tienen programador asignado y se encuentran en desarrollo:
- Creación manual de recibos desde la pestaña recibos de la factura.
- Verificación automática de DNI, NIF, etc.
- Nuevo modelo Ciudad.
- Revisión de las regularizaciones de impuestos para solventar los problemas encontrados.
- Nuevo informe de impuestos.
