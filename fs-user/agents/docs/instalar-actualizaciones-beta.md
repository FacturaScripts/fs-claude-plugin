Lanzamos **actualizaciones beta** de FacturaScripts y sus plugins antes de marcarlas como **estables**. De esta forma podemos hacer que solamente algunos usuarios prueben estas actualizaciones antes de lanzarlas para todo el mundo.

## ⚠️ BETA = pruebas
Que una actualización sea beta significa que está en fase de pruebas y **puede tener fallos todavía no detectados**. Use las beta bajo su propia responsabilidad.

## 🧪 Cómo activar las actualizaciones beta
Para activar las actualizaciones beta debemos marcar esa opción desde el menú administrador, panel de control. En la sección **por defecto** y en su grupo **Avanzado**:

![actualizacion beta](https://i.imgur.com/Hh5oaNH.png)

Una vez activada, pulsamos guardar y vamos al menú administrador, **actualizador** para ver las actualizaciones beta que hay disponibles.

### ⬇️ Cómo descargar una versión beta de FacturaScripts
Podemos descargar una versión beta del core desde esta url:
- https://facturascripts.com/DownloadBuild/1/beta
