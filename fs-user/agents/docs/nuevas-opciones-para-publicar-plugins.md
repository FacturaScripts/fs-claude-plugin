Desde hoy ya se pueden subir **plugins privados** para FacturaScripts 2018. Si vas a la **sección plugins** y pulsas el **botón nuevo**, verás que el asistente te permite ahora especificar el tipo de plugin:
- **Público**: todo el mundo puede verlo y descargarlo.
- **Privado**: solamente tú podrás verlo y descargarlo. Y quien tenga autorización.

## Tipos de plugins para la versión 2017
Para la versión 2017 los tipos son: externo, gratis, pago, regalo y oculto. Puedes cambiar tu plugin de tipo en cualquier momento, en la **sección editar**.

### Externo
Si marcas tu plugin como externo, lo puedes vender a través de **tu propia tienda online**. Simplemente indica el **precio** y el **enlace** (a la tienda) en la **sección editar** de tu plugin.

### Gratis o público
Los plugins gratuitos o públicos, como su nobre indica, son gratuitos, no hay que pagar nada. Todo el mundo puede verlos y descargarlos.

### Pago
Solamente un administrador puede marcar un plugin como plugin de pago. Se venden a través de esta web y están sujetos a una **comisión del 25%**. Si estás interesado, solicítalo a través de la sección contacto.

### Regalo
Todo el mundo puede verlos, pero solamente los usuarios registrados podrán descargalos.

### Oculto o privado
Solamente tú y los clientes autorizados podrán verlo y descargarlo.

## Claves de plugins
Para los plugins de **pago**, con cada compra se genera una clave para el cliente. Esta clave permite al cliente descargar el plugin y actualizarlo. Si el plugin es **externo** u **oculto** (privado), podrás **crear claves** manualmente desde la **sección claves** del plugin.

## Caducidad de las claves
Puedes establecer la caducidad de las claves de tu plugin desde la sección editar de tu plugin. Es el parámetro caducidad.

Espero que con esto os animéis a crear y publicar más plugins ;-)
