Esta es la lista de problemas más frecuentes con FacturaScripts y sus soluciones.

## 💻 Problemas tras la instalación o migración
Si tiene errores 404, 500 o errores de página en blanco tras la instalación, o tras mover la instalación a **otro servidor o hosting**, consulte la [lista de problemas frecuentes tras la instalación de FacturaScripts](/publicaciones/problemas-frecuentes-instalacion-641). También revise la [lista de requisitos de FacturaScripts](/publicaciones/requisitos-917), ya que el instalador comprueba los requisitos, pero si traslada la instalación, puede que el nuevo servidor no cumpla los requisitos y falta alguna de las extensiones de PHP necesarias.

## 🔑 He olvidado la contraseña
Si ha olvidado su contraseña, siga esta [guía para cambiar su contraseña de FacturaScripts](/publicaciones/he-olvidado-mi-contrasena). Es una guía muy completa, con los casos más comunes y todas las soluciones posibles.

### 🔒 Se ha bloqueado temporalmente el acceso desde su IP
Para evitar que cualquier software malicioso pueda robar sus datos, FacturaScripts bloquea automáticamente la IP cuando se fallan repetidas veces al introducir la contraseña. Siga la [guía de bloqueo de IPs](/publicaciones/bloqueo-de-ip-410) para resolver este problema.

## ✉️ Problemas con el email
Si tiene problemas al configurar el envío de emails, consulte la [documentación sobre el envío de emails en FacturaScripts](/publicaciones/envio-de-emails). Incluye ejemplos con los proveedores más comunes.

## 🚨 Fatal error #1
Si le aparece una pantalla de error fatal, es importante que pulse el **botón informar** para que nos llegue el informe de error y podamos solucionarlo. Además de esto y para que pueda seguir trabajando, la pantalla de error incluye un botón para desactivar todos los plugins. Si el error viene de un plugin, desactivando todos los plugins podrá continuar trabajando.

## ⚠️ El soporte para PHP 7.x terminará pronto
Si le aparece un mensaje de que el soporte para su versión de PHP terminará pronto. ¿Dónde tiene instalado FacturaScripts?

- Si está instalado en un hosting, vaya al panel de control del hosting, busque la sección de PHP y seleccione PHP 8.1
- Si está instalado en windows, haga una copia de seguridad, instale la nueva versión para windows y restaure la copia de seguridad.
