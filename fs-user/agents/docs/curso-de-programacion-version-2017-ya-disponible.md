Ya está disponible el curso básico de programación (versión 2017). Con este curso aprenderá a crear modelos, listados y formularios de edición en minutos, gracias las últimas mejoras que se incluyeron en la versión 2017.

## Lecciones
- Introducción.
- El entorno de programación.
- Su primer plugin.
- Estructura de un plugin.
- Prioridades de plugins.
- Controladores de listados.
- Controladores de edición.
- Los modelos.
- La estructura de las tablas.
- Herencia de modelos.
- Botones y acciones.
- Métodos de uso habitual en controladores.
- El sistema de actualización de plugins

Todos los miembros del equipo de desarrollo pueden obtener gratis este curso usando [este enlace](https://facturascripts.com/cursos/curso-basico-de-programacion-version-2017?action=promo&promocode=pruebas2k17dev).
