Los plugins son complementos que añaden nuevas funciones o modifican el comportamiento de FacturaScripts. Puede ver los plugins instalados haciendo clic en el **menú Administrador, Plugins**. Si se trata de una instalación nueva, no tendrá ninguno.

![lista plugins](https://facturascripts.com/MyFiles/2025/01/2503.png?myft=57db6a4123460bb4b5858a9c9f0b8fd045d8efd2)

Los plugins pueden estar **activos** o no, es decir, puede tener plugins instalados, pero inactivos. Para activar un plugin, pulse el botón activar en la lista de plugins.

![cómo activar un plugin](https://facturascripts.com/MyFiles/2025/01/2505.png?myft=efe008c8e90acad3631ad1033b9be77d914a32db)

## 🗃️ Plugins disponibles
La lista de plugins disponibles para FacturaScripts está en la [sección plugins](https://facturascripts.com/plugins) de esta web. Ahí encontrará **plugins gratuitos** y otros que son **de pago**. Si quiere descargar e instalar uno, tendrá que hacer clic en el plugin que quiera y después pulsar el botón de descargar. Si el plugin es de pago, primero tendrá que pulsar el botón de comprar, comprarlo y posteriormente podrá descargarlo.

## ➕ Instalar un plugin
Para instalar un plugin haga clic en el **botón añadir** y seleccione el archivo zip del plugin. El archivo zip de cada plugin lo puede obtener desde la [sección plugins](https://facturascripts.com/plugins) de esta web. Una vez instalado pulse el botón activar para activarlo.

![instalar un plugin en FacturaScripts](https://facturascripts.com/MyFiles/2025/01/2504.png?myft=8efbaf4d93cbb91c181ba3d6de2ae54ff36aa868)

### 🔍 ¿No aparece el botón añadir?
Si en el listado de plugins del menú Administrador, Plugins, no le aparece el botón añadir, es porque esa instalación tiene desactivada la instalación de plugins desde [el archivo config.php](https://facturascripts.com/publicaciones/el-archivo-config-php).

Servicios como **factura.city** desactivan esta opción porque son servicios que se ofrecen con plugins incluidos. Usted ya tiene una lista de plugins instalados, pero no puede añadir sus propios plugins. Contacte con el soporte de esa web para saber más.

### ⁉️ Instalo el plugin pero no aparece en la lista
Muchos hostings limitan el tamaño de los archivos que se pueden subir. En estos casos FacturaScripts siempre le avisa del tamaño máximo.

![plugin no se instala en FacturaScripts](https://facturascripts.com/MyFiles/2025/01/2506.png?myft=beb2a92b22d6189a0b941830ac05ae24b003d057)

Consulte con su hosting para que aumenten el tamaño o bien copie el plugin directamente a la carpeta Plugins de FacturaScripts.

### 🚫 Plugin incompatible
Cuando se presenta el mensaje de plugin incompatible puede ser por varios motivos:

![plugin incompatible](https://facturascripts.com/MyFiles/2025/01/2507.png?myft=ec4c1888dd4d17580d9c48251a70a11e8fd2d617)

- El archivo no es realmente un plugin de FacturaScripts.
- Ha copiado manualmente **el archivo ZIP** a la carpeta de Plugins de FacturaScripts en lugar de pulsar el **botón añadir** en el listado de plugins.
- Intenta instalar un **plugin de una versión obsoleta de FacturaScripts** en una versión superior.
- Intenta instalar un plugin nuevo en una **versión muy antigua de FacturaScripts**. Haga clic en el menú Administrador, Actualizador, y actualice.
