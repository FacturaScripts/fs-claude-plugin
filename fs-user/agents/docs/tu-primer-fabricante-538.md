Puedes ver el listado de fabricantes desde el **menú Almacén → Fabricantes**.

## 🛠️ Crear un fabricante

Para crear un nuevo fabricante, accede al **menú Almacén → Fabricantes**, y haz clic en el **botón nuevo**. Completa los campos que se solicitan:

- **Código:** Ingresa un código único para el fabricante.
- **Nombre:** Proporciona el nombre del fabricante.

Una vez completados estos campos, puedes proceder a asignar el fabricante a un artículo o producto según sea necesario.

## ⚖️ Diferencias entre fabricantes y proveedores

Los fabricantes permiten agrupar o filtrar productos, mientras que los proveedores son entidades a las cuales se les compran productos. Además, un producto puede tener solo un fabricante, pero puede ser adquirido de múltiples proveedores.
