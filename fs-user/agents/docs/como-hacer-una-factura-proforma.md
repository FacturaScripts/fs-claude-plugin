Las facturas proforma son una forma de presupuesto. Puedes [crear presupuestos de venta](/publicaciones/tu-primer-presupuesto-de-venta-351) fácilmente desde el menú Ventas → Presupuestos.

Una factura proforma es un documento informativo que se emite antes de una venta, con el objetivo de informar de los detalles al cliente.

## 🖨️ Nuevo modo de impresión avanzado
Uno de los aspectos que no estaba disponible en los presupuestos es la opción de imprimir una factura proforma.

Una de las ventajas de FacturaScripts es que puedes crear formatos de impresión personalizados, como el nuevo formato de &quot;factura proforma&quot;. Esto significa que ahora, entre las opciones de impresión, tienes el modo avanzado, que te permite elegir el formato e incluso el idioma. Una de las opciones que puedes seleccionar es la impresión del formato Proforma.
