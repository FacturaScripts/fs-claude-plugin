Puede actualizar tanto el núcleo como los plugins desde el **menú Administrador → Actualizador**.

![actualizar facturascript](https://i.imgur.com/eVTpaR6.png)

## 💻 Actualizar el núcleo
Si hay actualizaciones del núcleo, le aparecerá el componente CORE o CORE-2018 en la lista de actualizaciones. Pulse el botón descargar y después el botón actualizar.

![actualizar el core](https://i.imgur.com/K6CVnJW.png)

### 📅 FacturaScripts 2017
Si tiene instalada la versión 2017 y quere actualizar, consulte nuestra [guía sobre cómo actualizar desde FacturaScripts 2017](/publicaciones/actualizar-desde-facturascripts-2017-292).

## 🔌 Actualizar plugins
Para actualizar los plugins funciona exáctamente igual, si hay actualización, pulse el botón descargar y después el botón actualizar. Tenga en cuenta que debería **actualizar el núcleo** antes que los plugins.

### 💰 Actualizar plugins de pago
Para actualizar los plugins de pago necesita primero **registrar la instalación**, en caso contrario le saldrá un mensaje de `download-error` o `Código 401 - SUBSCRIPTION-NOT-FOUND` al intentar actualizar el plugin.

#### ®️ Registrar instalación
Para registrar la instalación pulse el **botón registrar** instalación en el actualizador. Tenga en cuenta que debería iniciar sesión en facturascripts.com para poder registrar la instalación.

![registrar instalacion](https://i.imgur.com/rRCNqxo.png)

Una vez registrada la instalación, solamente falta vincular la suscripción del plugin pulsando el **botón vincular**.

![vincular plugin de pago](https://i.imgur.com/CUQcvXN.png)

Tras vincular el plugin, ya puede actualizarlo sin problemas desde el actualizador. Tenga en cuenta que cuando compra un plugin, además del plugin, obtiene soporte de actualizaciones durante X meses. Cuando este soporte caduque recibirá un email informándole de los pasos a seguir.

## 📈 Telemetría
También es importante señalar que al registrar una instalación, se envía periódicamente a facturascripts.com información estadística como la versiones de PHP, versión de FacturaScripts, plugins activos, etc. A este proceso lo llamamos [telemetría](/publicaciones/telemetria-de-facturascripts).
