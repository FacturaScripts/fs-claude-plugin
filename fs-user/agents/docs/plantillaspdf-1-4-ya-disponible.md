Esta actualización soluciona los problemas detectados al imprimir textos con saltos de líneas, además de incorporar un **desglose completo de impuestos** y soporte para las **nuevas opciones de descuento** de FacturaScripts 2020.

## Nuevas opciones
- **Ocultar observaciones**: marcando esta opción no se imprimirán las observaciones de facturas, albaranes, pedidos o presupuestos.
- **Mostrar  numero2**: marcando esta opción, si el documento de venta tiene numero2 se mostrará al imprimir.

![nuevas opciones plantillaspdf](https://i.imgur.com/8lnJr3B.png)

## Desglose de impuestos
Ahora se muestra un desglose de impuestos si la factura (albarán, pedido o presupuesto) tiene más de un impuestos:

![desglose impuestos plantillaspdf](https://i.imgur.com/lZqFSnV.png)

## Novedades de FacturaScripts 2020
Esta actualización también soporta los [nuevos descuentos globales introducidos en FacturaScripts 2020](https://facturascripts.com/publicaciones/lo-nuevo-de-facturascripts-2020).
