En esta reunión hemos comentado los avances en el **cálculo de impuestos** de documentos de compra y venta. Ya están correctamente implementados el régimen de IVA de **recargo de equivalencia** y el exento, además del **IRPF**. Y también está operativa la configuración de **impuestos por zonas**, es decir, el poder configurar FacturaScripts para que aplique un impuesto u otro (o ninguno) en función de la dirección de facturación del cliente.

También se han comentado las mejoras en las interfaces de facturas, albaranes, etc. Ahora al visualizarlos se muestran etiquetas con información relevante:
- Qué usuario lo creó.
- Si se ha enviado por emil y cuando.
- Si está pagado.

## Dashboard
Definitivamente el Dashboard será sacado del núcleo esta semana y movido al plugin con el mismo nombre. Más adelante, cuando el desarrollo esté más avanzado, volverá al núcleo.

## Nueva beta
A finales de semana se lanzará una nueva beta con todos estos cambios. Y si no surge ninguna incidencia se migrará definitivamente la web a facturascripts.com

### Actualización de FacturaScripts 2017
Una vez sea migrada la web, se lanzará una nueva actualización de FacturaScripts 2017 para que pueda actualizarse con las nuevas urls. De no hacerlo dejarían de funcionar las actualizaciones en la versión anterior.
