Hemos lanzado nuevas versiones de [remote printer](https://facturascripts.com/remote-printer), la aplicación para imprimir tickets de FacturaScripts.

## Nueva beta
Junto al botón descargar de remote printer encontrarás ahora un botón **versión beta**. Ahí podrás descargar esta nueva versión con las siguiente nuevas características:
- Ahora se ejecuta directamente al **inicio de Windows**, sin necesidad de configuración.
- También se minimiza automáticamente a la **bandeja del sistema**.
- Evita imprimir líneas la típica línea en blanco cuando no hay tickets a imprimir.

### 2018
También hemos preparado una versión de remote printer específica para FacturaScripts 2018, que se puede usar junto al **plugin PrintTicket**.
