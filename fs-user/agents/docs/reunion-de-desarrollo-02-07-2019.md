Hoy, tras el lanzamiento de la [versión 2018.04](https://facturascripts.com/publicaciones/facturascripts-2018-04-ya-disponible) y la [OpenExpo Europe](https://facturascripts.com/publicaciones/retirada-de-plugins-y-novedades-para-julio), hemos comentado las novedades a incluir en la versión 2018.05, que llegará previsiblemente el día 8 de Julio.

## FacturaScripts 2018.05
Las novedades ya terminadas para esta versión son:
- Sistema de tarifas completado, compatible con el de 2017.
- Personalización de formato de impresión de facturas, albaranes, pedidos, etc.
- Revisión de las regularizaciones de IVA.
- Revisión de los principales controlador para añadir más información relevante. Ejemplo: ya se puede ordenar y buscar en todos los listados de contabilidad.

**Novedades por completar**:
- Generar los PDF acorde a la personalización del formato definida por el usuario para la factura, albarán, etc.
- Generar varios recibos manualmente para las facturas.
- Poder desactivar la ejecución de ciertas tareas del cron.

En desarrollo, pero sin fecha:
- [Crear informe de impuestos para poder sacar el desglose de impuestos en facturas de compra o de venta](https://facturascripts.com/EditTask?code=146).
