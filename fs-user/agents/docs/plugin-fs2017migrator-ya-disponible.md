Esta plugin, para **FacturaScripts 2018**, permite migrar los datos de 2017 a la nueva estructura.

## Funcionamiento
- Haga una **copia de seguridad** de la base de datos y archivos de FacturaScripts 2017.
- **Instale FacturaScripts 2018**, conectando a la base de datos que usase en la versión 2017.
- Instale el plugin FS2017Migrator.
- Haga clic en el **menú Administrador** &gt; **Panel de control** &gt; **Migrador 2017**.
- Haga clic en iniciar.
