Se agrego la impresión de tickets y ahora solo acepta cantidades mayores al total solo en los pagos en efectivo.

Para poder imprimir tickets:
* Tener instalado el [RemotePrinter](https://www.facturascripts.com/remote-printer) en su versión para facturascripts 2018.
* Tener instalado el plugin [TicketPrinter](https://www.facturascripts.com/plugins/PrintTicket).
