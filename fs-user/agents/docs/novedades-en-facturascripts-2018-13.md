Con esta actualización se introducen mejoras en la **exportación a PDF**:
- Las facturas de venta muestran ahora una sección &quot;**forma de pago**&quot; con los detalles de pago de la factura: sus recibos, la cuenta bancaria asociada, y vencimientos.
- Las facturas/albaranes/etc con más de un impuesto muestran ahora una sección con el **desglose de los impuestos**.
- Los archivos ahora usan el título como nombre del archivo.

La exportación a CSV y Excel también ha sido mejorada en esta actualización y ahora también usa el título del documento como nombre del archivo.

## Otros cambios
- Se ha añadido el campo orden a los valores de los atributos (los atributos de productos: **tallas y colores**). Así podemos ordenar los valores como queramos.
- También se han mejorado varias pantallas más para que se adapten mejor a tablets y smartphones.

### Mejoras para desarrollo
- Añadido [widget color](https://facturascripts.com/doc/1/desarrollo-sobre-facturascripts-2018/creacion-de-plugins/las-vistas-xml-xmlview/columns/widget/widget-color).
- Solucionado bug al añadir columnas a un grupo existente de un xmlview mediante extensiones.
- La barra de debug muestra ahora la diferencia en milisegundos entre las consultas a la base de datos.

## Plugin PlantillasPDF
Junto con este lanzamiento también stá disponible en la tienda el [plugin PlantillasPDF](https://www.facturascripts.com/plugins/plantillaspdf), con el que podemos **personalizar el diseño de facturas**, albaranes, etc.
