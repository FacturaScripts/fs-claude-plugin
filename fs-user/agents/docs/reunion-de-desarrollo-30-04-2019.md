En esta reunión hemos comentado las mejoras a desarrollar para la siguiente actualización. Principalmente en materia de agrupar documentos (presupuestos, pedidos y albaranes).

## Tareas asignadas
- [Tarea #119](https://facturascripts.com/EditTask?code=119): modificar la clase BusinessDocumentGenerator (permite generar documentos a partir de otros) para poder indicar propiedades distintas, por ejemplo una fecha distinta.
- [Tarea #120](https://facturascripts.com/EditTask?code=120): modificar los xmlview de los listados de presupuestos, pedidos y albaranes para añadir el botón de agrupar.
- [Tarea #116](https://facturascripts.com/EditTask?code=116): al insertar un nuevo registro con un EditController recargar la página para tener el ?code= siempre presente y evitar problemas.
- [Tarea #110](https://facturascripts.com/EditTask?code=110): modificar la clase InvoiceToAccounting para usar las subcuentas definidas en el impuesto, si lo están.
