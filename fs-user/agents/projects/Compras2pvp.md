Guarda como precio de venta los precios de compra en albaranes o facturas. Además establece un descuento máximo automático en las ventas de esos productos, es decir, al añadir el producto a una venta, se añade con el precio de compra y el descuento aplicado en la compra.

# ¿Y esto para qué sirve?
Para determinados negocios donde el proveedor establece el precio de venta. De esta forma al vender siempre tienes el precio que ha marcado el proveedor, y además tienes el descuento máximo que le puedes hacer al cliente (el que te ha hecho el proveedor). Lo habitual es dejar el descuento a 0%. Y si le quieres hacer descuento al cliente le pones 5% o 10%, pero sabes cuál es el descuento máximo que puedes hacer (el que te hizo el proveedor).
