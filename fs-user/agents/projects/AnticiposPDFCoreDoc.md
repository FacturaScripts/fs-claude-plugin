Complemento del plugin Anticipos:
- Da soporte para incluir el detalle de los anticipos recibidos, al generar el PDF del documento de Compra o de Venta mediante el CORE.
- NO funciona si se está utilizando el plugin PlantillasPDF.
- NO funciona si otro plugin ha personalizado el archivo PDFDocument del CORE.
