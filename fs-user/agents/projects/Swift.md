Swift es un plugin para FacturaScripts que añade la funcionalidad de mostrar el código Swift asociado a las cuentas bancarias en facturas y otros documentos donde se incluya el IBAN de la cuenta. Si la cuenta bancaria no tiene un código Swift configurado, el campo permanece vacío. Este complemento es ideal para empresas que trabajan con transferencias internacionales, ya que el código Swift es imprescindible para identificar entidades bancarias en el extranjero y garantizar transacciones seguras y rápidas.
