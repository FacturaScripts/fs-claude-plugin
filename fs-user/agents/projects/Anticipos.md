Plugin que permite generar cobros y pagos a cuenta. Se pueden vincular al cliente, al proveedor, al proyecto o al documento de compra o venta correspondiente. Para ello, al visualizar un cliente, proveedor, proyecto o documento de compra o venta (presupuesto, pedido o albarán) aparecerá la pestaña Anticipos, desde donde podremos generar un anticipo vinculado automáticamente a ese documento. Desde la factura solo se podrán visualizar los anticipos vinculados.
También se pueden generar anticipos desde su propio submenú (Ventas&gt;Anticipos y Compras&gt;Anticipos) . En este caso, solo los administradores tendrán acceso para poder vincularlo, posteriormente, a un documento.

Si necesitas adjuntar archivos a los anticipos, p.ej. el comprobante de pago, tienes este complemento que le da soporte:
- https://facturascripts.com/plugins/anticiposaddfile

Quizás te puedan interesar estos complementos del plugin, para incluir los anticipos en el documento de venta generado en PDF:
- https://facturascripts.com/plugins/anticiposplantillaspdfsalesdoc -&gt; Cuando se utiliza el plugin PlantillasPDF
- https://facturascripts.com/plugins/anticipospdfcoredoc -&gt; Cuando No se utiliza el plugin PlantillasPDF
