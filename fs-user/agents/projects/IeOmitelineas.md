Añade una opción que permite omitir la impresión de las lineas. Útil cuando estamos vendiendo un artículo compuesto, pero no queremos imprimir toda la información, sólo el total.
Ejemplo: Vendemos bicicletas montadas, y queremos facturar el trabajo total, no los componentes. Pero queremos controlar el stock de piezas y beneficios. En este caso añadiríamos una línea sin precio, ni cantidad ni iva para que quedara registrado, omitiríamos todas las líneas y listo.
**Requisitos:** Plugin PlantillasPDF
