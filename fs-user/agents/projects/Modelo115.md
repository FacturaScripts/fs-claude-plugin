El Modelo 115 es una declaración trimestral de las retenciones practicadas a los alquileres de inmuebles urbanos en los que desarrollamos una actividad empresarial. También incluye el Modelo 180 que es lo mismo pero anual.

# ¿Quién tiene que presentar el modelo 115 o 180?
Están obligados a presentar este modelo todos los autónomos, empresarios, o sociedades que paguen alquileres de oficinas o locales, siempre que sean inmuebles urbanos.

# ¿Cómo se rellena el modelo 115 o 180?
Instala el plugin y haz clic en el menú Informes → Modelo 115 y 180. Selecciona el ejercicio y trimestre.
