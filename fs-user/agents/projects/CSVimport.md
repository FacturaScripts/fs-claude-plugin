Permite importar clientes, proveedores, productos o artículos, familias, fabricantes y facturas desde archivos Excel o CSV.

Dispone de un modo de importación avanzada, donde podemos seleccionar a qué corresponde cada columna del archivo, y también dispone de varios perfiles ya programados, para importar datos de FacturaScripts, Facturaplus, Factusol, Holded, Contasimple, SAGE, etc ... sin tener que seleccionar a qué corresponde cada columna.

También permite dejar programadas las importaciones: seleccionas una url y cada cuanto quieres que se procese, y el cron se encarga de realizar esas importaciones periódicamente.
