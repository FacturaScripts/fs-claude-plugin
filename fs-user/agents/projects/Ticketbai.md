Implementación de Ticket-bai para FacturaScripts. Permite firmar electrónicamente las facturas de venta, enviarlas a la hacienda foral correspondiente (Álaba, Bizkaia o Guipúzcoa) y guardar el correspondiente código QR.

¿Qué es ticketbai?
Ticket BAI es el sistema de factura electrónica del país vasco y será de obligado uso para todos los negocios de allí en unos meses. El objetivo del sistema TBAI es luchar contra el fraude fiscal, además de:
- Garantizar la tributación de cada negocio o persona según sus ingresos reales.
- El máximo control de los ingresos en efectivo de los negocios.
- Que consumidor esté protegido recibiendo un ticket con el que poder reclamar sus derechos por la compra realizada.
- Disminuir los costes de las obligaciones fiscales.

¿Cómo funciona tiket bai en FacturaScripts?
Con este plugin todas las facturas de venta emitidas se envían automáticamente a la correspondiente hacienda foral, obteniendo un código QR que se añade a la factura a la hora de imprimir.

Requisitos:
- PHP 7.4 o superior con la extensión GMP activada.
- Debes registrarte en la hacienda foral correspondiente como software garante, ya que no se nos permite ofrecer licencias a terceros. La normativa de ticketbai no se pensó para software libre.
