TPVneo es el plugin que transforma FacturaScripts en un sistema de punto de venta (TPV) completo, rápido e intuitivo. Diseñado para tiendas, comercios, bares o negocios con venta directa, permite registrar operaciones, imprimir tickets y controlar la caja sin salir del entorno de FacturaScripts.

Con TPVneo puedes crear varios terminales, cada uno con su configuración propia: tipo de documento (factura o albarán), formas de pago, usuarios y permisos. Así podrás gestionar diferentes puestos o sucursales con total independencia, manteniendo los datos centralizados en una misma instalación.

La interfaz táctil está optimizada para mostradores y tablets, facilitando el trabajo diario. Además, ofrece funciones avanzadas como aparcar ventas, devoluciones rápidas, movimientos de caja (apertura, ingresos, retiradas y cierre), y la posibilidad de imprimir tickets o enviar facturas por email.

TPVneo se integra al 100 % con la contabilidad e inventario de FacturaScripts, garantizando un flujo de trabajo unificado, sin duplicidades ni errores. Es ideal para negocios minoristas, hostelería y servicios, donde la agilidad y el control del efectivo son esenciales.

- Permite configurar varios terminales, de forma que se pueda usar en varios puestos o tiendas distintas.
- Se puede configurar en cada terminal para generar facturas o albaranes.
- Las comandas o ventas se pueden aparcar para luego recuperar.
- Se pueden registrar movimientos de caja: sacar o meter dinero.
- Puede imprimir tickets, generar el PDF de las facturas o incluso enviarlo por email al cliente.

Terminal Punto de Venta (PoS) con diseño para pantalla táctil.
