Plugin de conciliación bancaria:
- Permite importar los movimientos bancarios (norma 43) y vincularlos a una cuenta bancaria.
- Incluye un asistente para vincular esos movimientos con recibos de facturas, de forma que se marquen las facturas como pagadas y quede reflejado cómo y a través de qué banco se hizo.
- También se pueden vincular los movimientos con remesas y asientos contables.

Requiere del plugin CSVimport.
