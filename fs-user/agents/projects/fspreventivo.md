Permite llevar el control de revisiones preventivas de instalaciones, edificios, etc.
Se configura el tipo de trabajo a revisar y las tareas que lleva cada tipo de trabajo.
Por ejemplo para una instalación de alarma:
 Tipo de trabajo: Comprobar centralita
Items a revisar: conectividad
            seguridad
            cableado

Una vez configurados los tipos de trabajo y sus tareas asociadas, se crearán órdenes de trabajo para las que se ha de marcar cada ítem a revisar como Bien, Mal o no revisado.
