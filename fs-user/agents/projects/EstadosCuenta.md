Este Plugin ayuda a generar un Estado de Cuenta por Cliente, según sea necesario, la idea es tener un documento con los datos necesarios para enviar al cliente y poder notificar sobre facturas que están pendientes de cobro o algunas que estén atrasadas, ahora se calculan los dias vencidos de forma automatica y se filtran las facturas no pagadas, asi como solamente las que tengan mas de 1 dia vencidas
