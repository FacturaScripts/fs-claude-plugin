Con este plugin puede crear plantillas de asientos para repetirlos rápidamente y ahorrar mucho tiempo con todos esos asientos que se repiten periódicamente, como nóminas, pagos, seguros, etc.

Para facilitar esta tarea FacturaScripts proporciona una serie de plantillas. Gracias a ellas se pueden introducir rápidamente asientos en el diario sin casi la intervención del usuario, por lo que no es necesario tener conocimientos de contabilidad y se reduce la posibilidad de cometer errores ya que tenemos cumplimentados la mayoría de los datos.
