Permite la generación de tickets para imprimirlos usando impresoras térmicas. Se usa en conjunto con la aplicación de impresión fsprinter https://facturascripts.com/remote-printer

El plugin te permite agregar de igual forma líneas con el texto que desees en el encabezado o pie del ticket, esto lo puedes hacer desde el menú 
Administrador-&gt;Tickets Ajustes.

Documentos soportados actualmente:
- Albarán de cliente.
- Factura de cliente.
- Pedido de cliente.
- Servicio de cliente.

Formatos de precios:
- Precion con iva incluido.
- Precio sin iva.
- Sin precios.

Recuerda que para poder utilizar el plugin tienes que configurarlo, puedes revisar la configuración en el siguiente enlace https://facturascripts.com/publicaciones/configuracion-del-plugin-printticket-37

Para poder imprimir tickets es necesario ejecutar fsprinter:
https://facturascripts.com/remote-printer
