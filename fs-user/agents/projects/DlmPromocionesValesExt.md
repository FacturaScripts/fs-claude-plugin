Con el Plugin DlmPromocionesValesExt podremos usar el plugin DlmPromocionesVales a través de TPVNeo.

Este plugin permitirá la creación de promociones aplicables a productos, familias de producto, establecimientos o empresas concretas dentro de nuestro ERP, e incluso controlar si estas promociones se aplican sobre un importe mínimo de venta o solo ciertos días de la semana.

Además se permitirá la venta de bonos o vales a nuestros clientes mediante las 2 referencias indicadas en la documentación, por el importe que deseemos. Este importe se irá reduciendo hasta llegar a 0€ conforme el cliente vaya usando el bono o vale en sus compras.
