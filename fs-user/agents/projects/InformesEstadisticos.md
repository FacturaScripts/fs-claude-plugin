Mediante este plugin para FacturaScripts podrá obtener informes estadísticos en formato PDF, en base a los datos de los documentos de compra y venta introducidos en la aplicación y según su ejercicio, pudiendo comparar los datos un ejercicio anterior.

Los informes se agrupan diferentes grupos según su origen o funcionalidad: 
    - Clientes: Ventas realizadas con márgenes respecto a otros ejercicios.
    - Proveedores: Compras realizadas a los proveedores.
    - Agentes: Ventas realizadas con márgenes respecto a otros ejercicios.
    - Productos: Venta de los productos con indicación del coste y beneficio obtenido.

Este nuevo apartado se añade al menú Informes con el nombre de &quot;Estadísticas&quot;.

IMPORTANTE:
1. Este plugin utiliza el plugin ExtendedReport para su correcto funcionamiento al generar los informes en PDF. Puede obtener el plugin de manera gratuita visitando la página del proyecto https://facturascripts.com/plugins/extendedreport.
2. Este plugin actualmente sólo genera PDF, por lo que no es posible exportar la información a otros formatos como xml o csv.
