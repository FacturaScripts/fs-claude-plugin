Este plugin para FacturaScripts está diseñado para facilitar la impresión de documentos de venta en impresoras de tickets térmicas, ofreciendo múltiples opciones de personalización para adaptarse a distintos tipos de negocio:

-Permite modificar el formato del ticket, agregando lineas personalizadas en el encabezado y los pie del ticket.

-Soporta distintos tipos de códigos de barras, ideales para identificar rapidamente los documentos.

-Configura el tamaño de fuente para adaptar el ticket a distintos modelos de impresora.

-Opciones de impresión con IVA incluido, desglosado o en modo ticket regalo (sin precios).

-Compatible con la aplicación TicketPrinter para Android, lo que permite imprimir desde un dispositivo móvil sin necesidad de PC.

Ideal para puntos de venta minoristas, restaurantes, tiendas de ropa, entre otros.


*** La App para Android, Widnows, MacOS es gratuita si quieres apoyar el desarollo puede hacerlo adquiriendo el plugin.
