Con este plugin podrá asignar comisiones de venta a sus agentes o comerciales y realizar la liquidación de las mismas.

Las comisiones se pueden asignar por agente, por cliente, por familia e incluso por producto, es decir, puede asignar:
- Una comisión del 3% para cualquier venta de un agente.
- Una comisión del 5% para un agente concreto.
- Una comisión del 10% para todas las ventas de productos de una familia.
- Una comisión del 15% para todas las ventas de un producto.
Y puede ajustar mejor cuando aplicar una comisión u otra con las prioridades. 

También permite establecer penalizaciones a los descuentos, de forma que podrá reducir el porcentaje de comisión de los agentes si aplican algún descuento al cliente.

Puede ver más detalles de cada una de las funciones del plugin en la pestaña documentación.
