Permite imprimir etiquetas de códigos de barras de productos. Desde almacén, albaranes o facturas. Además de poder generar códigos de barras para los productos que todavía no tienen.

Permite imprimir en impresoras normales con papel de A4 de etiquetas. Formatos:
- APLI 10490 (68 etiquetas - 48.5x16.9mm)
- 24 etiquetas (3x8 - 64.6x33.8mm)
- 21 etiquetas (3x7 - 63.5x38.1mm)

También permite imprimir en impresoras de etiquetas. Formatos:
- Brother 29x90mm
- 40x25mm
- 50x25mm
- 100x100mm

Opciones:
- Imprimir códigos de barra.
- Imprimir precios.
- Imprimir descripciones.
- Número de etiquetas por producto.

Requisitos:
PHP debe tener activadas las extensiones GD o imagemagick. Puede activar la extensión GD o la extension imagemagick desde el panel de control de su hosting, o desde el archivo php.ini de su servidor.

Cómo imprimir etiquetas de un albarán:
Abra el albarán y en la nueva pestaña etiquetas, pulse el botón imprimir.

Cómo generar códigos de barras:
Desde el listado de productos, en el menú almacén, productos, seleccione los productos para los que desee generar el código de barras, a continuación pulse el botón del código de barras.
