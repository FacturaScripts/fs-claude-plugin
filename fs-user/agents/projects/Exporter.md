Plugin que permite exportar y descargar en Excel la lista completa de productos, variantes, proveedores, clientes, contactos y facturas, independientemente de que sean miles o cientos de miles de registros.

Un cron genera estos archivos cada 30 minutos. De esta forma no se necesitan procesar cada vez que descargas.
