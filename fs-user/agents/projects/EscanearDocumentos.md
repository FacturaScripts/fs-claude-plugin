Permite escanear documentos (facturas, albaranes, tickets) para importarlos mediante OCR e Inteligencia Artificial. Perfecto para importar todas esas facturas que recibes en PDF o incluso los tickets de supermercados, tiendas, etc.

Archivos soportados:
- PDF
- JPG
- PNG

Este plugin requiere registrar la instalación de FacturaScripts para su funcionamiento y su uso está limitado a:
- 100 escaneos mensuales para patrocinadores ORO.
- 50 escaneos mensuales para patrocinadores PLATA.
- 25 escaneos mensuales para el resto.
