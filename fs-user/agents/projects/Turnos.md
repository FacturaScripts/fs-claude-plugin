Con este plugin puedes dar de alta nuevos turnos para tus clientes, editar la información existente y gestionar su estado de forma simple y ordenada.

Los turnos permiten definir una fecha, horario y duración, facilitando una planificación más precisa de la agenda y evitando solapamientos innecesarios.

Cuando se utiliza junto con el plugin Calendario, los turnos se integran automáticamente en una vista centralizada, permitiendo visualizar y organizar la agenda de forma clara dentro del calendario general del sistema.

La integración con Calendario también habilita el envío de notificaciones a los clientes, como confirmaciones y recordatorios, y añade campos de control para identificar si dichas comunicaciones fueron enviadas.

Puedes consultar la documentación para conocer los pasos necesarios para la integración y las opciones disponibles.
