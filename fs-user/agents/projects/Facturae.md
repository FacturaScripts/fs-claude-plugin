Añade soporte para la factura electrónica española: el formato facturae 3.2.

Este plugin añade la pestaña facturae a las facturas de cliente. Desde esta factura puede rellenar todos los campos necesarios para la administración, como la oficina contable, el órgano gestor, la unidad tramitadora y el órgano proponente. Una vez completado el formulario, puede seleccionar su certificado, su contraseña y firmar la factura.

Códigos almacenados:
Al rellenar los códigos de oficina contable, organo gestor, etc, el plugin recordará los códigos de la última factura de ese cliente, por lo que no es necesario almacenarlos, ni recordarlos.

Integración con face.gob.es:
El plugin también le permite enviar la factura directamente al portal face.gob.es

Compatibilidad con la facturacion electronica de otros países:
Este plugin NO ES COMPATIBLE con los sistemas de factura electronica de otros países. Solamente es válido para España.
