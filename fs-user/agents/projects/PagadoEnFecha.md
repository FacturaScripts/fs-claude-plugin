El plugin PagadoEnFecha para FacturaScripts es una herramienta esencial para optimizar la gestión de pagos en tu negocio. Con este plugin, puedes seleccionar múltiples recibos de compras y ventas y definir una fecha de pago específica. Esto permite cambiar el estado de los recibos de manera masiva y precisa, ahorrando tiempo y reduciendo errores manuales.

Añade un botón en la pestaña de Recibos, tanto de Facturas de Compras como de Facturas de Ventas, donde indicar la fecha de pagos permite seleccionar múltiples recibos a la vez.
