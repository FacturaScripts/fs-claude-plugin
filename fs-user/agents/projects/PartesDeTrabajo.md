Con este plugin podrá conectar FacturaScripts con la aplicación de Partes de Trabajo en la Nube:
- Podrá ver los partes de trabajo directamente desde FacturaScripts.
- Podrá importar los partes como albaranes en FacturaScripts.

Tenga en cuenta que debe tener contratado el servicio de Partes de Trabajo en la nube para poder usar este plugin. En concreto, el plan de 18€ al mes (o superior). Y debe pedir la clave de API a info (arroba) partedetrabajo.com

Si desea más información:
- https://partedetrabajo.com
