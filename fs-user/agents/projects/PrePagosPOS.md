Este plugin integra el plugin PrePagos y POS para FacturaScripts y automatiza el proceso para registrar los pagos anticipados (prepagos) al registrar ventas a través del plugin POS. Al realizar una venta en el POS, si el plugin PrePagos está habilitado y el documento generado no es una factura, se crea automáticamente un prepago relacionado con la venta, el plugin no necesita configuración ya que solo realiza el registro de los pagos.
