FaeArg - Facturación Electrónica Argentina para FacturaScripts


¿Querés emitir comprobantes electrónicos en Argentina sin complicaciones? Con FaeArg podés hacerlo directamente desde FacturaScripts con solo un botón para generar y otro para anular.

Este plugin complementa FacturaScripts y otorga una solución completa de facturación electrónica, 100% compatible con los requerimientos de ARCA/AFIP.
Solo tenés que cargar tu factura como siempre… y presionar el botón. El resto se hace automáticamente: se valida con AFIP, se obtiene el CAE, se genera el PDF oficial y se guarda en tu factura, listo para imprimir, enviar o descargar.
Además, incluimos un formato profesional de comprobante PDF, con el diseño ya preparado y adaptado a los requerimientos oficiales (incluye QR, CAE, y tus datos con logo).

También podés consultar y validar los CAEs ya generados, ideal para auditoría y control interno.
Compatible con Facturas A, B y C, notas de crédito, productos, servicios y cualquier combinación.
No necesitás conocimientos técnicos ni certificados digitales propios. El plugin delega toda la conexión con AFIP de forma segura y confiable. Y si lo preferís, te ayudamos personalmente a dejar todo configurado.

Funcionalidades destacadas:

Botón para generar el CAE
Botón para anular (con nota de crédito automática)  con motivo para auditorias
PDF profesional incluido, con QR y datos del comprobante
Consulta y validación de CAEs ya emitidos
Soporte para Facturas A, B, C y notas de crédito
Sin instalaciones técnicas ni certificados: todo resuelto
Asistencia personalizada si la necesitás


Requisitos para usar el plugin FaeArg:

PHP 8.2 o superior
Es indispensable contar con PHP 8.2 o versiones más recientes para asegurar el funcionamiento estable del sistema.

Delegación del servicio de facturación
Este plugin funciona delegando la generación de comprobantes electrónicos a un servicio externo autorizado.
Para ayudarte con esta configuración, puedes revisar la guía paso a paso.
Además, si preferís asistencia personalizada, podés contactarnos y te ayudamos a dejarlo listo.
El proceso es seguro y transparente: tus facturas se validan en línea, se obtiene el CAE, se genera el PDF oficial y se devuelve a tu sistema para su registro.

Punto de venta habilitado para facturación electrónica (AFIP)
Debés tener un punto de venta habilitado en AFIP con WebService (WSFE) activo. Si aún no lo tenés, podés generarlo desde tu cuenta de AFIP con clave fiscal.

Configuración de decimales en FacturaScripts
Es necesario configurar tu instalación con máximo 3 decimales para asegurar compatibilidad con los requisitos de validación de AFIP. Esto evita errores al validar importes.

Logo con tamaño moderado
El PDF oficial generado incluirá tu logo. Para una correcta visualización, te sugerimos usar una imagen de ancho no mayor a 800 píxeles. Cuanto más grande sea el archivo, más pequeño se verá en el comprobante.
