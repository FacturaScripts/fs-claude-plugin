Este plugin ayuda a agregar campos nuevos a los formularios de compras y de ventas, de momento se pueden agregar campos en el header, footer, y las lineas.

El Plugin en si mismo no crea los campos, crea un plugin nuevo con el campo a agregar el plugin genera codigo que agrega campos a la base de datos, por lo mismo se recomienda utilizar en un entorno de pruebas y al concretar todas las pruebas el plugin resultante llevarlo a produccion.

se parece mucho al plugin de CRUDgenerator
