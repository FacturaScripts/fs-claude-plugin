Permite generar facturas, albaranes, pedidos o presupuestos (de compra o de venta) que se repiten cada x tiempo, ya sean días, semanas, meses o años. La forma más fácil de hacer compras o ventas periódicas.

Imagina que cada mes haces la misma factura al mismo cliente, ahora podrás crearla una sola vez. Ve a ventas &gt; recurrentes, pulsa el botón nuevo, selecciona el tipo factura, el cliente, cantidad 1, plazo meses, fecha de inicio, introduce las líneas y listo. Cada mes se generará una factura para ese cliente con las líneas indicadas.

Opciones destacadas:
- Podemos indicar una fecha fin, ideal para cuando sólo queremos generar facturas hasta cierta fecha.
- Podemos indicar un precio en cada línea o dejarlo en blanco y que se use el precio actualizado.
- Podemos indicar un descuento por línea.
- Podemos indicar una descripción personalizada e incluso usar códigos como {FECHA}, {ANYO}, {MES} o {NOMBREMES}.

Generación automática:
Las facturas, albaranes, pedidos o presupuestos que indiquemos se generarán automáticamente cuando llegue la fecha indicada si tenemos el cron configurado. O bien podemos generarlas cuando quedamos pulsando el botón generar pendientes en los listados, o el botón generar en la ficha del documento recurrente.

En definitiva la mejor opción para todos los que buscan hacer una facturación pediódica.
