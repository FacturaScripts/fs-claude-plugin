Plugin para FacturaScripts que permite imprimir documentos (presupuestos, pedidos y albaranes) sin mostrar el IVA en los totales, dejando únicamente el valor neto como total. Especialmente útil para empresas que necesitan proporcionar documentos sin desglose de impuestos para ciertos tipos de operaciones o clientes.
