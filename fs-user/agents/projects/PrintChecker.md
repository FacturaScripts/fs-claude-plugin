Plugin de Control de Impresión para Documentos

Este plugin proporciona una solución eficaz para aquellos que manejan la impresión regular de documentos como facturas y albaranes. Su función principal es evitar confusiones y agilizar el proceso de identificación de documentos que ya han sido impresos.

Características principales:

1. Campo de Estado de Impresión: Se añade un campo distintivo en el listado de albaranes y facturas, permitiendo a los usuarios visualizar rápidamente el estado de impresión de cada documento.
2. Actualización Automática: Al imprimir un documento, este se marca automáticamente como &quot;Impreso&quot;, asegurando que los usuarios estén siempre al tanto de qué documentos han sido procesados.

Con este plugin, garantizas un flujo de trabajo más organizado, evitando repetir impresiones y facilitando la gestión documental.

Nota: Por ahora esta hecho solo para documentos de venta.
