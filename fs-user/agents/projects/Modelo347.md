El Modelo 347 de la hacienda española es un modelo informativo para la declaración anual de operaciones con terceros. Instale este plugin para obtener los datos necesarios para rellenar el modelo.

# ¿Quién está obligado a presentar el modelo 347?
Todas aquellas personas físicas o jurídicas que desarrollan actividades comerciales con terceras personas por un importe superior a 3005,06 euros.

# ¿Cómo presentar el modelo 347?
Instale el plugin y haga clic en el menú Informes → Modelo 347. Seleccione el ejercicio y pulse el botón previsualizar. En la parte inferior encontrara los desgloses por clientes y por proveedores. Pulse el botón descargar para obtener el archivo excel que puede usar para enviar a hacienda o a su gestor.

# Otras utilidades
Este modelo también es interesante para saber a cuantos clientes ha realizado ventas en un ejercicio determinado o a cuantos proveedores a comprado. Dado que puede modificar el importe mínimo, lo puede poner a 0 euros para obtener un listado de todos los clientes que le han comprado algo en el último año.
