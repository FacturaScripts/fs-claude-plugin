El plugin TPVneo_Vales amplía la funcionalidad del plugin TPVneo, permitiendo la gestión eficiente de vales durante las devoluciones de clientes. Al realizar una devolución, se emite automáticamente un vale que se imprime con un código de barras junto al ticket original. Este vale puede ser canjeado en futuras compras, utilizándose como un método de pago.

El sistema registra detalladamente todos los vales emitidos, incluyendo la fecha de emisión, el usuario responsable, el terminal, y la caja desde la que se generó. Además, cuando un vale es utilizado, se almacena información como la fecha y hora del canje, el documento utilizado (factura o ticket), el usuario que lo aplicó, y el terminal y caja correspondientes. También permite que el usuario emita sus propios vales de manera manual.

Adicionalmente, el plugin añade un código de barras a los tickets con el número de ticket, facilitando así la búsqueda y gestión de documentos.
