Añade la opción de trazabilidad al plugin de Producción.
Los datos de trazabilidad deben introducirse mediante el proceso de creación de órdenes de trabajo. Una vez introducido el parte a cada una de los productos a producir se les puede indicar los datos de trazabilidad.

Se necesita tener los plugins de Producción y de Trazabilidad para poder utilizar/activar este plugin.
