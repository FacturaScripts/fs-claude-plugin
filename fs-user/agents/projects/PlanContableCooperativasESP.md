Sustituye el Plan Contable por defecto que trae Facturascripts,  por el plan contable para pymes adaptado para las cooperativas españolas, con las subcuentas de 10 dígitos y la configuración inicial de las cuentas especiales.

NOTA: Se debe instalar antes de importar el plan contable en Facturascripts.

Cuando creas un ejercicio, a continuación debes importar el plan contable. Para ello, pulsa el botón &quot;importa el plan contable&quot;, y si tienes instalado y activado este plugin, no selecciones ningún archivo y pulsa aceptar para que te cargue el plan de cuentas para cooperativas.

Los idiomas disponibles son castellano y catalán. Se instalará la versión correspondiente al idioma seleccionado (castellano o catalán) en tu instalación.
