Añade un sistema de notificaciones y avisos tanto en el programa como por email, para estar siempre informado de los cambios y ejecuciones del programa. Permite suscribirse a los procesos crons, así como a los procesos en segundo plano. Por ejemplo, puede recibir notificaciones cuando se modifique una factura, cuando se añada un cliente nuevo, o elimine algún dato.

También permite avisos y notificaciones de stock bajo, stock máximo, riesgo alcanzado del cliente, presupuestos/pedidos/recibos caducados, actualizaciones pendientes y más.

Además, se incluye un chat interno entre usuarios para poder hablar entre los empleados y departamos de la empresa.

Y para programadores, este plugin está diseñado para que otros plugins puedan añadir sus notificaciones de forma fácil y rápida.
