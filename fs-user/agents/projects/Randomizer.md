Permite crear datos de pruebas para usar en demos o pruebas. Con este plugin puede generar datos aleatorios de clientes, proveedores, productos, familias, fabricantes, albaranes, pedidos, presupuestos...

ADVERTENCIA:
Estos datos generados no se separan del resto. No utilice este plugin pensando que podrá deshacer todo lo generado con un clic.
